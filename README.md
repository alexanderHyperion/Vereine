# Vereinssoftware – Skateclub Burgau e.V.

Webbasierte Vereinsverwaltungssoftware für den Skateclub Burgau e.V.

## Technologie
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express
- **Datenbank:** PostgreSQL
- **Hosting:** Hetzner (Nürnberg, Deutschland)

## Module
- Mitgliederverwaltung
- Finanzverwaltung & SEPA-Lastschrift
- Veranstaltungen & Protokolle
- Anträge & Schriftverkehr
- Kalender (öffentlich einbettbar)

## Setup
Siehe `backend/.env.example` für Konfiguration.
